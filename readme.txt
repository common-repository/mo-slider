Widget image slider using post ID.

== Description ==

Display an image slider using post IDs in WordPress. A WordPress widget plugin that displays a slider from specific posts. This plugin will display a slider in any widget area. The slider will show the featured image, title and excerpt. The slider is responsive; visibly clean display and functionality on any device (mobile or desktop).

= Features =
* Display responsive post slider on phone, tablet and desktop
* Displays the post featured image, title and excerpt 
* Lightweight and loads quickly
* [BXSLIDER](https://bxslider.com/ "BXSLIDER")

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
